Untuk bisa masuk ke aplikasi OpenPos, Anda membutuhkan akun pengguna. Akun pengguna terdiri dari:

1. Username (nama pengguna), misalnya admin, kasir, joko, dll.
2. Password (kata sandi).

Akun pengguna akan dibagikan setelah pembayaran.

Setelah [membuka alamat url toko Anda](/panduan/cara-menggunakan-aplikasi), akan tampil sebuah form login, masukkan username dan password Anda pada form tersebut.

Jika salah maka akan muncul peringatan.

Jika benar maka halaman dashboard dari aplikasi OpenPos akan dibuka.

## Macam-Macam Pengguna

Ada 4 jenis penguna dalam aplikasi OpenPos:

1. **Admin**, memiliki akses ke semua fitur, kecuali penjualan dan shift.
2. **Manager**, seperti Admin, tetapi tidak bisa melihat laporan laba.
3. **Gudang**, memiliki akses ke fitur-fitur pergudangan.
4. **Kasir**, memiliki akses ke fitur penjualan dan shift.

Setiap jenis pengguna dapat memiliki banyak pengguna, misalnya **Kasir** ada kasir 1, kasir joko, dst. Pengguna bisa ditambahkan oleh Admin di menu [kelola pengguna](/panduan/mengelola-pengguna).
