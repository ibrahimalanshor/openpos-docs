Aplikasi OpenPos dapat digunakan di semua perangkat, seperti handphone, tablet, laptop dan komputer.

Aplikasi OpenPos tidak perlu diinstal di perangkat, karena aplikasi OpenPos berbasis web, sehingga hanya perlu diakses melalui browser.

Untuk menggunakan aplikasi OpenPos, Anda perlu menginstal satu aplikasi saja, yaitu browser. Contoh browser:

- Google Chrome
- Mozilla Firefox
- Microsoft Edge
- Safari
- Browser Bawaan Perangkat
- dll

Jika sudah ada browser, selanjutnya ikuti panduan berikut:

- [Cara Membuka Aplikasi](/panduan/cara-membuka-aplikasi)
