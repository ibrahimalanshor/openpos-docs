Pengaturan adalah fitur untuk mengatur data kelengkapan toko, seperti nama toko, logo dan alamat.

Untuk memulainya, buka menu `Pengaturan` di menu samping kiri.

Setelah terbuka, akan ditampilkan form detail toko seperti nama, logo, dll. Lengkapi semuanya.

Setelah lengkap klik tombol simpan.
