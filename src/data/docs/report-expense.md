Laporan pengeluaran adalah laporan yang berisi riwayat pengeluaran operasional dalam periode tertentu (hari, bulan, tahun).

Untuk membuat laporan pengeluaran, klik `Menu Laporan > Laporan Pengeluaran` di menu samping kiri.

Setelah terbuka, pilih periode laporan, tanggal/bulan/tahun.

Lalu klik tombol `Tampilkan`.

Akan muncul laporan pengeluaran sesuai periode yang dipilih.

Di bagian atas terdapat informasi total pengeluaran dan jumlah pengeluaran.

Di bawahnya terdapat tabel daftar pengeluaran pada periode yang dipilih.

Untuk mendownload laporan menjadi file PDF, klik tombol `Download PDF`.

Untuk mendownload laporan menjadi file Excel, klik tombol `Download Excel`.
