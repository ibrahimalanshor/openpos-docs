Pengeluaran operasional adalah fitur untuk mencatat pengeluaran operasional pada toko.

Pengeluaran operasional digunakan untuk komponen perhitungan laba bersih.

Contoh pengeluaran operasional:

- Bayar listrik
- Bayar wifi
- Gaji karyawan
- dll
