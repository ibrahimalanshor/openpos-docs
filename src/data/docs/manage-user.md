Pengguna adalah orang yang bisa menggunakan aplikasi OpenPos.

Ada empat jenis pengguna di aplikasi OpenPos:

1. **Admin**, mengelola semua fitur.
2. **Manager**, mengelola semua fitur kecuali laporan laba.
3. **Kasir**, membuat penjualan, shift dan uang kas.
4. **Gudang**, mengelola stok, membuat restok dan stok opname.
