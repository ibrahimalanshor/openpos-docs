---

Untuk mengubah password pengguna, buka menu `Pengguna` di menu samping kiri.

Pilih pengguna yang ingin diubah passwordnya di tabel daftar pengguna, lalu klik tombol edit.

Akan muncuk _popup_ yang berisi form detail pengguna, di bagian input password, masukkan password yang baru.

Setelah lengkap, klik tombol `Simpan`.
