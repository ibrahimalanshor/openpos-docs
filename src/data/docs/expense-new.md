Untuk mencatat pengeluaran operasional, buka menu `Pengeluaran` di menu samping kiri.

Setelah terbuka, klik tombol `+ Pengeluaran`.

Masukkan jumlah pengeluaran, nama pengeluaran, dan deskripsi. Contoh:

- Jumlah: 1.800.000
- Nama: Gaji Karyawan
- Deskripsi: Gaji karyawan bulan mei 2025

Setelah dimasukkan semua, klik tombol `Simpan`.

Setelah berhasil tersimpan, data pengeluaran akan masuk ke tabel riwayat pengeluaran.
